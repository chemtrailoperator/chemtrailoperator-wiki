# Endgame Alpha General Information

About the Plugin: This plugin is a content plugin for Endless Sky. It allows the player to take part in an Alpha conflict as a Navy officer. As a navy officer, you gain access to exclusive ships, outfits, and jobs.

Game Requirements: As of December 2025, compatible with Endless Sky v0.10.16

Story line requirements to begin: Completed the Free Worlds Main Plot This can be checked in your game save with the line "event: free worlds main plot completed"

Recommendations: Be experienced with ship combat and have plenty of money available.

Save your game before installing the plugin. There are some events in the game that have a large effect on system and planet accessibility, and story to revert those has not yet been finished.

First mission: To test if the plugin is working, visit any Free Worlds spaceport and eventually you will receive a conversation popup about Raven in a TMBR hoodie.
